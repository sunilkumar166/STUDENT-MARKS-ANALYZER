# Student Marks Analyzer

This is a Python-based mini project that helps analyze student marks in a simple and interactive way.

## Features
- Take marks of multiple students  
- Display all marks  
- Calculate total marks  
- Calculate average marks  
- Find highest and lowest marks  
- Separate even and odd marks  
- Check pass and fail status  

## Concepts Used
- Input and Output  
- If-Else Conditions  
- For Loop and While Loop  
- Lists  
- Logical and Arithmetic Operators  

## How to Run
1. Install Python  
2. Download or clone this repository  
3. Run the file using:

```bash
python student_marks_analyzer.py
